NIRGAM is a systemC based discrete event, cycle accurate simulator for research in Network on Chip(NoC). It provides substantial support to experiment with NoC design in terms of routing algorithms and applications on various topologies. http://nirgam.ecs.soton.ac.uk/ 

NIRGAM v2 provide much greater capabilities and behave more accurate.

To start with NIRGAM read "NIRGAM_manual.pdf" or browse "docs" directory.
